Basic cmd command:
用于记录日常用到的cmd命令

1. 查看端口情况
```
netstat -ano
```
2. 查看进程号
```
netstat -ano|findstr "端口号"
```

4. 查看IP地址
```
ipconfig
```
5. 查看网络包情况
```
ping
```

6. 自行查看帮助
```
help
```

7. 画图
```
mspaint
```
8. 笔记本
```
notepad
```

9. 删除服务
```
sc delete mysql
```
